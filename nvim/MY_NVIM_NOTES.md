# keybindings

## key settings
<leader>: \
<esc>: jj
za: fold

## jump to buffer/tab
<leader> + n: Jump to buffer n
<leader> + Fn: Jump to tab n

## Fn
<F1>: open this note
<F3>: format
<F11>: nohls
<F12>: nvim-tree

## REPL and terminal
<leader> + t: terminal
<leader> + r: open REPL
<leader> + w: REPL send line
<leader> + e: REPL send block

## nvim-tree
a: Create file or directory
r: Rename file or directory
d: Delete file or directory
c: Copy
x: Cut
p: Paste
H: Toggle hidden files
